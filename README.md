# WABOT MULTI Web

Bundle ini khusus untuk versi web saja.

## File yang dipakai

- `multi_user_web.cjs`
- `public_multi/`
- `package.json`
- `package-lock.json`
- `.replit`
- `DEPLOY_REPLIT.md`
- `pesan.txt`

## Jalankan lokal

```bash
npm install
npm start
```

Buka:

```text
http://127.0.0.1:3000
```
